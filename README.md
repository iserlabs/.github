# .github

Org-wide defaults and brand for **Iser Labs**.

- [`profile/README.md`](profile/README.md) — renders on the org landing page, [github.com/iserlabs](https://github.com/iserlabs)
- [`brand/`](brand/) — logo, wordmarks, avatars, favicons, palette ([brand/README.md](brand/README.md))
- [`docs/`](docs/) — design system handoff (brand kit source)
- [`renovate-config.json`](renovate-config.json) — org-wide inherited Renovate config. **Dormant until activated:** install the Renovate (Mend) app on the org and enable **inherited config** (Mend cloud org setting, or `RENOVATE_INHERIT_CONFIG=true` self-hosted). Keeps every site's `@iserlabs/web-kit` git-tag pin current.

Community-health files placed here (CODE_OF_CONDUCT, CONTRIBUTING, issue templates)
apply org-wide as fallbacks for any repo that lacks its own.
